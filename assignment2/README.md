# QR Code Attendance System (With Alternative Web UI Bridge)

This subsystem runs a dynamic scanner using visual buffers to decode matrix symbols, cross-reference student lists, log active timestamps, and generate raw data sheets.

## Local Web Execution (Telegram Ban Workaround)
1. Navigate to `assignment2`
2. Run `npm install`
3. Launch via: `node server.js`
4. Access the interface: `http://localhost:3000`

## Production Bot Execution
Run via: `node bot.js`
